- 친구가 풀어보래서 풀었다.

- 만약 단어에서 매핑하려는 문자가 이미 매핑되었다면!
  - 매핑된 문자가 패턴의 문자와 일치하는지 확인한다.
  - 일치하면 다음 글자로
  - 다르면 return False
- 단어에서 처리중인 문자는 처음 보는건데 매핑할 패턴 문자는 이미 사용했다면!
  - return False
- 두쪽 다 클린하다면 새로운 매핑을 만들고 다음 글자로.
